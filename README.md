### Installation

CSS-Sprite-Maker requires some dependencies to run.

Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://github.com/veerbal1/CSS-Sprite-Maker.git
$ cd CSS-Sprite-Maker
$ npm install
$ npm start
```
